How To Run 'PageRank':
	The 'PageRank' is implemented in Java.
	Java SE 8 environment is required (it cannot compile under former versions).
	executable jar file has been exported, named 'PageRank.jar'.
	
	Use the following to run the program:
		cat input-file | java -jar PageRank.jar > output-file
